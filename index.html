Part 3: Grammar Focus

Simple Past Tense (Regular Verbs)

We use the simple past tense to talk about actions that finished in the past. For regular verbs, we add **-ed** to the base form.

Rule 1: Most verbs

Add **-ed** to the base form.

  • Example: work → worked
  • Example: help → helped

Rule 2: Verbs ending in -e

Add only **-d** to the base form.

  • Example: celebrate → celebrated
  • Example: live → lived
